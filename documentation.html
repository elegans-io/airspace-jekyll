How to integrate ORAC in your system

ORAC produces campaigns based on the information provided by the behavior of visitors on your website and/or by the past purchase of your users.

Online behavior

After you have contacted us, we will generate the KEY_ID to be used in the JavaScript snippet like the one below.


<!-- ORAC -->
<script type='text/javascript'>
  function setupORAC () {
    window.ORACObject = 'orac';

// check whether the Orac object is defined
  if (!('orac' in window)){

    // define the Analytics object
    window.orac = function(){

      // add the tasks to the queue
      window.orac.q.push(arguments);

    };

    // create the queue
      window.orac.q = window.orac.q || [
      ['create', 'ORAC-KEY_ID', 'auto'],
      ['send', 'surf']
    ];

    window.orac.l = new Date().getTime();
  }
</script>

<script type='text/javascript' src='//www.elegans-orac.io/behavior.js' async></script>
<!-- End ORAC -->

  • Greetings
  • History
  • Recommendations

  • Greetings

    {greetings}
    Different greetings expressions (Dear {name}, Hello...).

    {farewell}
    Farewell expressions (Bye, Regards...)

  • History

    {surf_history}
    Array of last visited object (pages, items...). Object can be defined by the customer, e.g.

    {surf_history.items}
    Last visited products (with name, url, image, price...)

    {surf_history.pages}
    Visited pages (with title, url...)

  • Recommendations

    {recommendations}
    Array of recommended object. As for history, object are defined by the customer, e.g.

    {recommendations.items}
    Recommended items (with .name, .url, .image, .price...)

    {recommendations.users}
    E.g. for social networks, users you might know (with title, url...)
